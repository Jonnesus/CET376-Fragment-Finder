# CET376 Fragment Finder
 
Arcade racing game with an adaptive difficulty system

---
## Controls
### Keyboard
Movement: WASD\
Jump: Space\
Crouch: Control

### Gamepad
Movement: Left Thumbstick\
Jump: South Button\
Crouch: East Button
